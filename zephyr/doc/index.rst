Zephyr OOT Module OMX Documentation
###################################

.. only:: release

   Welcome to the documentation for Zephyr OOT module OMX version |version|.

   Documentation for the latest (master) development branch of
   Zephyr OOT module OMX
   can be found at
   https://docs.zephyrproject.org/

.. only:: (development or daily)

   **Welcome to the Zephyr OOT module OMX's documentation
   for the master tree under development** (version |version|).

Use the version selection menu on the left to view
documentation for a specific version of Zephyr OOT module OMX.

For information about the changes and additions for releases, please
consult the published :ref:`zephyr_oot_module_omx_release_notes` documentation.

The Zephyr OOT module OMX is provided under the `Apache 2.0 license`_
(as found in the LICENSE file in the project's `GitHub repo`_).
The Zephyr OOT module OMX also imports or reuses packages, scripts,
and other files that use other licensing, as described in
:ref:`Zephyr_OOT_Module_OMX_Licensing`.

.. _Apache 2.0 license:
   https://github.com/zephyrproject-rtos/zephyr/blob/master/LICENSE

.. _GitHub repo: https://github.com/zephyrproject-rtos/zephyr
